## 📖 Sobre o Projeto

**Inovateca** é uma interface web de uma **Biblioteca Virtual** desenvolvida em HTML e CSS, com foco em experiência do usuário, navegação fluida e organização por categorias.

O projeto apresenta:

* 🔐 Sistema de Login e Cadastro
* 📚 Catálogo de livros organizado por categorias
* 👤 Área "Meus Livros" (emprestados e comprados)
* 🔎 Sistema de busca
* 🎥 Tutoriais e Central de Ajuda
* 🛡️ Área de Segurança Online
* 🎫 Sistema de abertura de chamado

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts (Poppins)
* Font Awesome (ícones)

## 🔐 Funcionalidades

### ✅ 1. Login

* Campo de e-mail
* Campo de senha
* Botão de acesso com rolagem suave para o catálogo

### ✅ 2. Cadastro

* Nome completo
* E-mail
* CPF
* Telefone
* Senha e confirmação

## 🚀 Como Executar

1. Baixe os arquivos do projeto
2. Coloque todos na mesma pasta
3. Abra o arquivo `index.html` no navegador

Não é necessário servidor ou banco de dados (projeto estático).
